### Sensing-Dev SDK

We have released the Sensing-Dec SDK installer [v24.01.04](https://github.com/Sensing-Dev/sensing-dev-installer/releases/tag/v24.01.04) which includes

* PyGObject for Python3.11
* **Aravis 0.8.30 (GenDC applied)** with aravis python
* **ion-kit 1.6.0** with ionpy + Halide
* OpenCV 4.5.5 (Optional)
* **WinUSB installer v24.02.02**

Updates on installer are:

* Added CICD tests for installer
* Added WinUSB installer's install command
* Made OpenCV optional on installer
* Added License file for SDK installer
* Removed unneccesary library files for runtime (Aravis dependencies)

Updates on features are:

* usb3vision: make async libusb default
* Fixed log-generation issue
* Updated ion-kit API by its major release
* Updated ionpy to ion-python and started officieal support on PyPI
* Made Runtime OpenCV optional
