### Sensing-Dev SDK

We have released the Sensing-Dec SDK installer [v23.11.01](https://github.com/Sensing-Dev/sensing-dev-installer/releases/tag/v23.11.01) which includes

* PyGObject for Python3.11
* Aravis 0.8.25 (GenDC applied) with aravis python
* **ion-kit 0.3.6** with ionpy + Halide
* OpenCV 4.5.5
* WinUSB installer

Updates are:

* Fixed the issue on framecount
* Fixed the issue on Gain/ExposureTime 0 value 
* Adding log feature
* Removed PixelFormat from input port of U3V BB
* Set up pip PyPI for ionpy

Details are in [ion-kit 0.3.5](https://github.com/fixstars/ion-kit/releases/tag/v0.3.5) and [ion-kit 0.3.6](https://github.com/fixstars/ion-kit/releases/tag/v0.3.6)
