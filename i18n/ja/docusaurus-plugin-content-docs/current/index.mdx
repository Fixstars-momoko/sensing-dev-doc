## Sensing-Devとは？

**Sensing-Dev**は、[**USB3 Vision (U3V) カメラ**](https://www.automate.org/a3-content/usb3-vision-standard)のユーザーをサポートするプロジェクトです。ソフトウェアパッケージ、チュートリアル、ディスカッションフォーラムを提供します。

U3Vカメラは**GenICam標準**に基づいており、カメラデバイスを簡単に制御できますが、プロジェクトをどこから始めれば良いかわからないことがあるかもしれません。
このウェブページは、U3Vカメラへの**アクセス**とデバイスからの**センサデータ**の取得に役立ちます。

このソフトウェアパッケージは、すべて**オープンソースソフトウェア**およびライブラリから構成されており、研究および商業利用での画像処理アプリケーションの開発が可能です。

## ゴール

私たちのゴールは、以下のソフトウェアスタックに基づいた一般的かつプラットフォーム非依存の画像処理フレームワークを使用して、U3Vカメラのユーザーが簡単に自分のマシンビジョンプロジェクトを開発できるようサポートすることです。
