# Installation - Linux 

This document describes the steps to install the software development environment for U3V camera on Linux.

## Content
* SDK Installation
  * [C++ User](#cpp)
  * [Python User](#python)

## C++ Userã€€<a id="cpp"/>

### Prerequisite
* OS: ubuntu 22.04 **with sudo privilege**
* Development environment
    * C++
* Hardware:
    * USB3.0 compatible USB port x2


### Sensing-dev Software Packages

#### Download the setup script

<pre>
<code class="language-bash">
wget -O setup.sh {this_version.latest_setup_URL}
</code>
</pre>

#### Install and setup SDK

Note that `--install-opencv` is optional.

The following command install SDK and its dependencies

<pre>
<code class="language-bash">
sudo bash setup.sh --install-opencv {this_version.linux_version}
</code>
</pre>

### Set Environment Variables

<pre>
<code class="language-bash">
export LD_LIBRARY_PATH=/opt/sensing-dev/lib:/opt/sensing-dev/lib/x86_64-linux-gnu:$LD_LIBRARY_PATH
</code>
</pre>

:::info Confirmation
Using `/opt/sensing-dev/bin/arv-tool-0.8` may help to check if the software package is appropriately installed. Please check [this page](../external/aravis/arv-tools) to know the procedure.
:::

## Python User <a id="python"/>
### Prerequisite
* OS: ubuntu 22.04 **with sudo privilege**
* Development environment
    * Python >=3.10
* Hardware:
    * USB3.0 compatible USB port x2

### Install dependency
<pre>
<code class="language-bash">
sudo apt install -y libgirepository1.0-dev gcc libcairo2-dev pkg-config python3-dev gir1.2-gtk-4.0<br />
pip3 install -U pip<br />
pip3 install pycairo<br />
</code>
</pre>

### Install Python Modules

Install aravis-python and gendc-python and ion-contrib-python to fully utilize

<pre>
<code class="language-bash">
pip3 install PyGObject<br />
pip3 instal aravis-python<br />
pip3 install ion-contrib-python=={this_version.ion_python_version}<br />
pip3 install gendc-python=={this_version.gendc_python_version}<br />
</code>
</pre>
